# WxCC Extensibility Guide

An interactive, multi-page developer guide for Webex Contact Center (WxCC) API extensibility.

## Pages

| Page | Description |
|------|-------------|
| [index.html](index.html) | Overview & Architecture |
| [getting-started.html](getting-started.html) | Auth, OAuth, tokens, first API call |
| [rest-apis.html](rest-apis.html) | REST APIs — Tasks, Agents, Queues, Config |
| [webhooks.html](webhooks.html) | Webhooks & Real-Time Events |
| [desktop-sdk.html](desktop-sdk.html) | Agent Desktop JS SDK & Custom Widgets |
| [flow-designer.html](flow-designer.html) | Flow Designer & Intelligent Routing |
| [reporting.html](reporting.html) | Reporting & Analytics APIs |
| [faq.html](faq.html) | FAQ — 30 Q&As with search |

## Features
- Webex brand design (Navy #00476B + Teal #00BCEB)
- Interactive: tabs, accordions, expand/collapse, copy-to-clipboard
- SVG architecture diagrams + Mermaid.js flow/sequence charts
- Annotated JavaScript code samples with syntax highlighting
- Mobile responsive with hamburger nav
- Scroll progress bar, back-to-top, scroll-spy sidebar

## Usage
Open `index.html` in any browser — no build step, no dependencies.
