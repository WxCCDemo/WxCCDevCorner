What is Webex Contact Center?

Webex Contact Center (WxCC) is Cisco's cloud-native, AI-powered contact center platform. It handles inbound and outbound customer interactions across voice, chat, email, and social channels — and exposes a rich set of APIs so developers can integrate it deeply into business systems.

Extensibility means you don't have to accept WxCC as a black box. You can:

  • Pull data from WxCC into your CRM, analytics platform, or reporting dashboards via REST APIs
  • React to contact-center events in real time using webhooks
  • Embed custom UI panels right inside the agent desktop using the Desktop JS SDK
  • Build intelligent routing flows and integrate external services through the Flow Designer
💡

Base URL

All WxCC APIs are hosted under https://webexapis.com/v1/. Authentication uses OAuth 2.0 access tokens.

5-Minute Quick Start

Get a list of all agents in your WxCC org in under 5 minutes.

1

Get an Access Token

Log in at developer.webex.com, go to your profile, and copy your personal access token (valid 12 hours) or create a Service App for production.

2

Find Your Org ID

Call GET /v1/organizations — the id field is your Org ID, needed for most WxCC endpoints.

3

Make Your First Call

Use the code below to list agents in your contact center.

⬡ JavaScript / Node.js
const axios = require('axios');

const TOKEN = 'YOUR_ACCESS_TOKEN';  // from developer.webex.com
const ORG_ID = 'YOUR_ORG_ID';

async function listAgents() {
  const response = await axios.get(
    `https://webexapis.com/v1/contactCenter/agents`,
    {
      headers: {
        'Authorization': `Bearer ${TOKEN}`,
        'Content-Type': 'application/json'
      },
      params: { orgId: ORG_ID, max: 50 }
    }
  );

  const agents = response.data.items;
  console.log(`Found ${agents.length} agents:`);
  agents.forEach(agent => {
    console.log(`  • ${agent.name} (${agent.email}) — Status: ${agent.status}`);
  });
}

listAgents().catch(console.error);
$ cURL
curl -X GET \
  "https://webexapis.com/v1/contactCenter/agents?orgId=YOUR_ORG_ID&max=50" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json"
✅

Expected Response

A JSON object with an items array. Each item contains the agent's id, name, email, and current status.
