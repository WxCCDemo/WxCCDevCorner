Reporting & Analytics APIs

Pull real-time queue stats, historical call data, agent performance metrics, and CDRs into your own dashboards, BI tools, or data warehouses.

On This Page
  • 📋 Data Types
  • 🔄 Data Pipeline
  • ⚡ Real-Time Stats
  • 📅 Historical Reports
  • 📞 Call Detail Records
  • 📤 Scheduled Export
  • 🖥️ Dashboard Architecture

Reporting APIs require the cjp:analyzer_read scope. Historical data is available for up to 13 months. Real-time data is a point-in-time snapshot — poll every 15–30 seconds for live wallboards, and combine with webhooks for instant state change notifications.

📋

What You Can Query

⚡

Real-Time Stats

Live queue depths, agent states, and active contact counts. Point-in-time snapshot refreshed on each API call.

📅

Historical Reports

Aggregated metrics by hour, day, week, or month. SLA compliance, AHT, abandon rates — up to 13 months back.

📞

Call Detail Records (CDR)

One row per contact with full timeline: queue time, talk time, hold, wrap-up, outcome, and call variables.

👤

Agent Performance

Per-agent stats: contacts handled, average handle time, adherence, and state duration breakdowns.

Real-Time Statistics

REAL-TIME QUEUE SNAPSHOT
async function getQueueSnapshot(queueId) {
  const { data: s } = await axios.get(
    `https://webexapis.com/v1/contactCenter/queues/${queueId}/statistics`,
    { headers: { Authorization: `Bearer ${token}` }, params: { orgId: ORG_ID } }
  );
  return {
    queueName:       s.queueName,
    contactsWaiting: s.contactsInQueue,
    longestWait:     s.longestContactInQueue, // seconds
    agentsAvailable: s.agentsAvailable,
    agentsOnContact: s.agentsOnContact,
    agentsIdle:      s.agentsIdle,
    serviceLevel:    s.serviceLevelPercentage // % in SLA window
  };
}

const snapshot = await getQueueSnapshot('QUEUE_ID');
console.table(snapshot);
AGENT STATE SUMMARY
async function getAgentStates(teamId) {
  const { data } = await axios.get(
    'https://webexapis.com/v1/contactCenter/agents/states',
    { headers: { Authorization: `Bearer ${token}` }, params: { orgId: ORG_ID, teamId } }
  );
  const summary = { Available: 0, Idle: 0, InACall: 0, WrapUp: 0 };
  data.items.forEach(a => { summary[a.agentState] = (summary[a.agentState] || 0) + 1; });
  return summary;
}

const states = await getAgentStates('TEAM_ID');
if (states.Available === 0) {
  await sendAlert('⚠️ No agents available in team!');
}
LIVE WALLBOARD CLASS
class LiveWallboard {
  constructor(queueIds, intervalMs = 30_000) {
    this.queueIds   = queueIds;
    this.intervalMs = intervalMs;
  }

  start() {
    this.refresh();
    this.timer = setInterval(() => this.refresh(), this.intervalMs);
  }

  stop() { clearInterval(this.timer); }

  async refresh() {
    const stats = await Promise.all(this.queueIds.map(id => getQueueSnapshot(id)));
    this.emit('update', stats);
  }

  emit(event, data) {
    // Push to your UI via WebSocket, EventSource, or DOM update
    document.dispatchEvent(new CustomEvent(`wallboard:${event}`, { detail: data }));
  }
}

const board = new LiveWallboard(['QUEUE_1', 'QUEUE_2', 'QUEUE_3']);
board.start();

Historical Reports

Query aggregated metrics for any time range up to 13 months back at hourly, daily, weekly, or monthly granularity.

JavaScriptYesterday's Queue Metrics → CSV Export
const yesterday = new Date(Date.now() - 86_400_000);
const dateStr   = yesterday.toISOString().split('T')[0]; // "2024-01-14"

const { data } = await axios.get(
  'https://webexapis.com/v1/contactCenter/reports/queues',
  {
    headers: { Authorization: `Bearer ${token}` },
    params: {
      orgId:       ORG_ID,
      startDate:   `${dateStr}T00:00:00Z`,
      endDate:     `${dateStr}T23:59:59Z`,
      granularity: 'HOURLY',  // HOURLY | DAILY | WEEKLY | MONTHLY
      metrics:     'contactsHandled,avgHandleTime,serviceLevelPercentage,abandonRate'
    }
  }
);

const report = data.items;
const totalHandled = report.reduce((sum, r) => sum + r.contactsHandled, 0);
console.log(`Total contacts handled yesterday: ${totalHandled}`);

// Export to CSV
const csv = [
  'Queue,Hour,Handled,AHT(s),SLA%,Abandon%',
  ...report.map(r =>
    `${r.queueName},${r.hour},${r.contactsHandled},${r.avgHandleTime},${r.serviceLevelPercentage},${r.abandonRate}`
  )
].join('\n');
require('fs').writeFileSync('report.csv', csv);
console.log('✅ report.csv written');

Call Detail Records (CDR)

One record per contact — the most granular data available. Full timestamps, agent assignments, and outcomes.

FieldTypeDescription
callIdstringUnique contact identifier
callerIdstringCustomer phone number or chat ID
agentIdstringHandling agent's ID
queueIdstringQueue where contact was routed
talkDurationintegerActive talk/chat time in seconds
holdDurationintegerTotal hold time in seconds
wrapUpCodestringAgent's selected outcome code
callVariablesobjectFlow/CAD variables attached to contact
JavaScriptPaginated CDR Fetch
async function fetchCDRs({ startDate, endDate, channelType }) {
  const allRecords = [];
  let cursor = null;

  do {
    const params = { orgId: ORG_ID, startDate, endDate, channelType, max: 1000 };
    if (cursor) params.cursor = cursor;

    const { data } = await axios.get(
      'https://webexapis.com/v1/contactCenter/recordings/calls',
      { headers: { Authorization: `Bearer ${token}` }, params }
    );

    allRecords.push(...data.items);
    cursor = data.next ? new URL(data.next).searchParams.get('cursor') : null;

  } while (cursor);

  return allRecords;
}

const cdrs = await fetchCDRs({
  startDate: '2024-01-01T00:00:00Z',
  endDate:   '2024-01-31T23:59:59Z',
  channelType: 'telephony'
});

const avgHandle = cdrs.reduce((s, c) => s + c.totalDuration, 0) / cdrs.length;
console.log(`Fetched ${cdrs.length} CDRs | Avg handle: ${Math.round(avgHandle)}s`);

Scheduled Data Export

For large-scale pipelines, schedule nightly exports to your data warehouse rather than polling continuously.

JavaScriptNightly ETL → BigQuery
const cron     = require('node-cron');
const { BigQuery } = require('@google-cloud/bigquery');

const bq = new BigQuery({ projectId: 'your-gcp-project' });

// Run at 2 AM every night
cron.schedule('0 2 * * *', async () => {
  const yesterday = new Date(Date.now() - 86_400_000);
  const dateStr   = yesterday.toISOString().split('T')[0];

  const cdrs = await fetchCDRs({ startDate: `${dateStr}T00:00:00Z`, endDate: `${dateStr}T23:59:59Z` });

  if (cdrs.length > 0) {
    await bq.dataset('contact_center').table('call_detail_records').insert(
      cdrs.map(r => ({
        call_id:       r.callId,
        caller_id:     r.callerId,
        agent_id:      r.agentId,
        queue_id:      r.queueId,
        date:          dateStr,
        talk_duration: r.talkDuration,
        hold_duration: r.holdDuration,
        total_duration: r.totalDuration,
        wrap_up_code:  r.wrapUpCode,
        channel_type:  r.channelType
      }))
    );
    console.log(`✅ Exported ${cdrs.length} CDRs to BigQuery`);
  }
});
