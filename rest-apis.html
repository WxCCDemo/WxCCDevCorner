🔌 REST APIs

Contact Center REST APIs

Manage every aspect of your Webex Contact Center programmatically — tasks, agents, queues, teams, and configuration. All endpoints use JSON and Bearer auth.

📞 Task / Contact APIs

A Task in WxCC represents a single customer interaction (call, chat, email). These APIs let you query, update, and manage active and historical tasks.

GET /v1/contactCenter/tasks List all active tasks in your org ▼

Returns paginated list of all currently active contact tasks. Filter by queue, agent, channel type, or status.

Query Parameters

ParameterTypeDescription
orgIdstringRequired. Your organization ID
channelTypestringFilter: telephony | chat | email | social
statusstringFilter: active | parked | held
maxintegerMax results per page (default 100, max 1000)
⬡ JavaScript
const response = await axios.get('https://webexapis.com/v1/contactCenter/tasks', {
  headers: { Authorization: `Bearer ${token}` },
  params: {
    orgId: ORG_ID,
    channelType: 'telephony',
    status: 'active',
    max: 50
  }
});

// Each task item:
// { id, channelType, status, queueId, agentId, callerId, startTime, ... }
const tasks = response.data.items;
console.log(`Active calls: ${tasks.length}`);
GET /v1/contactCenter/tasks/{taskId} Get details of a specific task ▼

Fetch full details of a task including call variables, routing history, and agent assignment.

⬡ JavaScript
const taskId = 'abc123-task-id';
const response = await axios.get(
  `https://webexapis.com/v1/contactCenter/tasks/${taskId}`,
  { headers: { Authorization: `Bearer ${token}` } }
);
const task = response.data;
console.log(`Task ${task.id}: ${task.channelType} — Agent: ${task.agentId}`);
POST /v1/contactCenter/tasks/{taskId}/end End / wrap up a task ▼

Programmatically end a task and set wrap-up reason.

⬡ JavaScript
await axios.post(
  `https://webexapis.com/v1/contactCenter/tasks/${taskId}/end`,
  { wrapUpCode: 'RESOLVED', notes: 'Customer issue resolved' },
  { headers: { Authorization: `Bearer ${token}`, 'Content-Type': 'application/json' } }
);

👤 Agent APIs

Manage agent profiles, retrieve state information, and assign agents to teams or skill profiles.

GET /v1/contactCenter/agents List all agents in the org ▼
⬡ JavaScript
async function listAgents(siteId = null) {
  const params = { orgId: ORG_ID, max: 200 };
  if (siteId) params.siteId = siteId;

  const response = await axios.get('https://webexapis.com/v1/contactCenter/agents', {
    headers: { Authorization: `Bearer ${token}` },
    params
  });

  return response.data.items.map(a => ({
    id: a.id,
    name: a.name,
    email: a.email,
    state: a.agentState,        // Available | Idle | InACall | WrapUp
    team: a.teamName,
    site: a.siteName
  }));
}

const agents = await listAgents();
const available = agents.filter(a => a.state === 'Available');
console.log(`${available.length} agents currently available`);
POST /v1/contactCenter/agents Create a new agent ▼
⬡ JavaScript
const newAgent = await axios.post(
  'https://webexapis.com/v1/contactCenter/agents',
  {
    orgId: ORG_ID,
    firstName: 'Jane',
    lastName: 'Smith',
    email: 'jane.smith@company.com',
    siteId: 'site-id-here',
    teamId: 'team-id-here',
    skillProfileId: 'skill-profile-id',
    dialNumber: '+14085551234',
    agentType: 'AGENT'   // AGENT | SUPERVISOR
  },
  { headers: { Authorization: `Bearer ${token}`, 'Content-Type': 'application/json' } }
);
console.log(`Created agent: ${newAgent.data.id}`);

📋 Queue APIs

Queues (Entry Points in WxCC) define how contacts are routed to agents. These APIs let you read queue stats and manage queue configuration.

⬡ List Queues
const response = await axios.get('https://webexapis.com/v1/contactCenter/queues', {
  headers: { Authorization: `Bearer ${token}` },
  params: { orgId: ORG_ID }
});
// Returns: id, name, channelType, serviceLevel, maxWaitTime, agentCount
response.data.items.forEach(q => {
  console.log(`Queue: ${q.name} | Agents: ${q.agentCount} | SLA: ${q.serviceLevel}%`);
});
⬡ Real-time Queue Stats
// Real-time queue statistics (snapshot)
const stats = await axios.get(
  `https://webexapis.com/v1/contactCenter/queues/${queueId}/statistics`,
  { headers: { Authorization: `Bearer ${token}` }, params: { orgId: ORG_ID } }
);
const s = stats.data;
console.log(`
  Queue: ${s.queueName}
  Contacts waiting: ${s.contactsWaiting}
  Longest wait:     ${s.longestWaitTime}s
  Agents available: ${s.agentsAvailable}
  Calls handled:    ${s.contactsHandled}
`);
⬡ Update Queue Settings
await axios.put(
  `https://webexapis.com/v1/contactCenter/queues/${queueId}`,
  {
    orgId: ORG_ID,
    name: 'Support Queue - Tier 1',
    serviceLevel: 80,          // % of calls answered within SLA time
    serviceLevelThreshold: 30, // seconds
    maxWaitTime: 300,          // max 5 min wait before overflow
    routingType: 'SKILLS_BASED'
  },
  { headers: { Authorization: `Bearer ${token}`, 'Content-Type': 'application/json' } }
);

⚙️ Show: Bulk-create teams from a list
▼
⬡ Bulk Create Teams
const teamNames = ['Billing', 'Tech Support', 'Sales', 'Escalations'];

async function bulkCreateTeams(siteId, teamNames) {
  const results = [];
  for (const name of teamNames) {
    const res = await axios.post(
      'https://webexapis.com/v1/contactCenter/teams',
      { orgId: ORG_ID, siteId, name, teamType: 'INBOUND' },
      { headers: { Authorization: `Bearer ${token}`, 'Content-Type': 'application/json' } }
    );
    results.push({ name, id: res.data.id });
    await new Promise(r => setTimeout(r, 200)); // gentle rate-limit spacing
  }
  return results;
}

const teams = await bulkCreateTeams('SITE_ID', teamNames);
console.table(teams);

Pagination

All list endpoints use cursor-based pagination. Use the next URL from the response to fetch subsequent pages.

⬡ Auto-paginate all results
async function getAllPages(url, params, headers) {
  const allItems = [];
  let nextUrl = url;
  let pageParams = { ...params, max: 500 };

  while (nextUrl) {
    const response = await axios.get(nextUrl, { headers, params: pageParams });
    allItems.push(...(response.data.items || []));

    // WxCC returns a 'next' link header for cursor pagination
    const linkHeader = response.headers['link'];
    const nextMatch = linkHeader?.match(/<([^>]+)>;\s*rel="next"/);
    nextUrl = nextMatch ? nextMatch[1] : null;
    pageParams = {};  // next URL already has params baked in
  }
  return allItems;
}

// Fetch ALL agents across all pages
const allAgents = await getAllPages(
  'https://webexapis.com/v1/contactCenter/agents',
  { orgId: ORG_ID },
  { Authorization: `Bearer ${token}` }
);
console.log(`Total agents: ${allAgents.length}`);
